# 🔬 تشخيص أمراض الجلد — Skin Disease Classifier

تطبيق ذكاء اصطناعي مبني بـ **Streamlit** يستخدم نموذج CNN مدرَّب على مجموعة بيانات **ISIC 2019** لتصنيف أمراض الجلد وتقديم توصيات طبية مخصصة.

---

##  الأمراض التي يشخصها التطبيق

| الكود | الاسم بالعربي |
|-------|--------------|
| MEL   | سرطان الجلد الميلانيني |
| BCC   | سرطان الخلايا القاعدية |
| SCC   | سرطان الخلايا الحرشفية |
| AK    | التقران الشعاعي (الحرشفي) |
| BKL   | التقران الدهني الحميد |
| DF    | الورم الليفي الجلدي |
| NV    | الشامة الميلانينية (وحمة) |
| VASC  | الآفات الوعائية الجلدية |

---

##  مميزات التطبيق

-  **رفع صورة** للمنطقة المصابة والحصول على التشخيص فوراً
-  **خريطة الجسم التفاعلية** — تحديد موقع الإصابة بدقة
-  **أسئلة شخصية** (السن، مدة الحالة، التاريخ العيلي، التعرض للشمس)
-  **درجة الخطورة** — خطير / متوسط / بسيط
-  **توصيات طبية** مخصصة لكل مرض
-  **توصيات إضافية** مبنية على إجابات المريض
-  **نسب جميع الأنواع** بشكل مرئي

---

##  طريقة التشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تأكد من وجود ملف الموديل
ضع ملف الموديل `ImranIsicModel.keras` في **نفس مجلد** `app.py`

```
 المشروع/
├── app.py
├── ImranIsicModel.keras   ← مهم جداً
├── requirements.txt
└── README.md
```

### 3. تشغيل التطبيق
```bash
streamlit run app.py
```

ثم افتح المتصفح على العنوان:
```
http://localhost:8501
```
